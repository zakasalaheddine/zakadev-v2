Bid Buddy is a modern and user-friendly web application that allows individuals to list products for auction and receive competitive bids from interested buyers. Built with a powerful tech stack, Bid Buddy offers a seamless and engaging bidding experience.

### Key Features:

- Product Listing: Users can easily share details and images of their products, setting an end date for the bidding process.
- Bidding System: Interested buyers can place their bids on listed products, with the highest bid at the end of the bidding period securing the purchase.
- Real-time Updates: The application provides real-time updates on bid amounts, ensuring transparency and fairness throughout the bidding process.
- User Authentication: Bid Buddy leverages NextAuth for secure user authentication, ensuring that only registered users can participate in bidding activities.
- Responsive Design: The application is built with Tailwind CSS, ensuring a modern and responsive user interface that adapts seamlessly to various devices and screen sizes.
- Notifications: Bid Buddy integrates with Knock, a notification service, to keep users informed about new bids, outbid events, and other critical updates, ensuring they never miss out on important bidding activities.

### Tech Stack:

- Front-end: Next.js, a React framework for server-side rendering and static site generation, ensuring optimal performance and SEO.
- Back-end: PostgreSQL, a robust and scalable relational database management system, paired with Drizzle ORM for efficient data management.
- Styling: Tailwind CSS, a utility-first CSS framework, providing a highly customizable and efficient styling solution.
- Storage: Cloudflare, a global content delivery network, providing secure and reliable storage for user-uploaded product images.
- Notifications: Knock, a powerful notification service for sending real-time updates to users across various platforms.


Bid Buddy offers a unique and engaging platform for individuals to buy and sell products through a competitive bidding process. With its modern tech stack and user-friendly interface, the application delivers a seamless and enjoyable experience for both buyers and sellers.
